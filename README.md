## âœ¨ Features

1.  **Chrono-Input (Landing Page)**:
    *   Validates user birthdate.
    *   Stores data in `sessionStorage` for cross-page persistence.
    *   Game-like entrance animations.

2.  **Mission Control (Dashboard)**:
    *   **Parallel Data Fetching**: Uses `Promise.all()` to fetch data simultaneously from:
        *   **Numbers API**: Year and Date trivia.
        *   **Open-Meteo**: Historical weather data for London on your birthdate.
        *   **ZenQuotes/History**: Historical events on your birthdate.
    *   **Chrono-Quiz**: A dynamically generated question based on your birth year.
    *   **Visual Archive Generator**: Creates a downloadable image with your Year Fact overlaid on a random artistic background using the HTML5 Canvas API.

3.  **Temporal Forecast (Predictor)**:
    *   Reveals 3 random future fortunes from a pool of 20+ outcomes.
    *   Interactive card-flip animations.
